# DeskFlow AI

DeskFlow AI, sirket icinde musteri, proje, gorev ve belge sureclerini yonetmek icin gelistirilen WPF masaustu uygulamasidir.

## Asama 1

Bu ilk adimda WPF uygulamasinin temel acilis akisi kuruldu:

- `App.xaml` uygulamanin baslangic noktasini belirler.
- `MainWindow.xaml` ilk ekrani tanimlar.
- `MainWindow.xaml.cs` ekrandaki buton tiklaninca calisan C# kodunu icerir.
